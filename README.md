### 截图识别工具

#### 👀 简介
一个能够通过拖动鼠标确定截图区域并识别其中文字内容的工具。

#### 🌟 特性
- 使用纯Java代码调用PaddleOCR。
- 集成Onnx推理引擎，支持在CPU上运行。

#### ⚡ 快速开始
- 下载并解压缩ocr.zip文件，然后运行ocr.exe。
- ⚠️ 建议将屏幕缩放比例设置为100%以确保最佳效果。
